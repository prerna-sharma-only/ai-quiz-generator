# 🤖 AI Quiz Generator Web App

An AI powered quiz platform that dynamically generates MCQ quizzes on any topic using LLM API.

## 🚀 Features

- Dynamic quiz generation using AI
- Topic selection & custom topic support
- Timer based quiz attempt
- Score analytics with performance circle
- Futuristic neural UI design
- Floating tech particle background
- Cursor glow interaction

## 🛠 Tech Stack

- Node.js
- Express.js
- HTML / CSS / JS
- Groq LLM API

## ▶️ How to Run
npm install
node server.js

Open:
http://localhost:3000
