## ✨ Overview

An **AI-powered quiz generation web application** that dynamically creates **MCQ-based quizzes on any topic** using a Large Language Model (LLM).

Users can enter a custom topic and instantly receive a **timed quiz experience** with real-time evaluation and performance insights.

---

## 🚀 Key Features

* 🤖 AI-based dynamic quiz generation
* 📝 Custom topic input support
* ⏱️ Timer-based quiz system
* 📊 Score analytics with performance visualization
* 🎨 Futuristic neural UI design
* ✨ Floating particle background
* 🖱️ Interactive cursor glow effects
